8:T1914,<h2>题目描述</h2>
<p>给定一个二叉搜索树, 找到该树中两个指定节点的最近公共祖先。</p>
<p>百度百科中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”</p>
<p>例如，给定如下二叉搜索树: root = [6,2,8,0,4,7,9,null,null,3,5]</p>
<p><img src="/images/leetcode/leetcode-235.png" alt="leetcode-235"></p>
<p>示例 1:</p>
<pre><code>输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 8
输出: 6
解释: 节点 2 和节点 8 的最近公共祖先是 6。
</code></pre>
<p>示例 2:</p>
<pre><code>输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 4
输出: 2
解释: 节点 2 和节点 4 的最近公共祖先是 2, 因为根据定义最近公共祖先节点可以为节点本身。
</code></pre>
<p>说明:</p>
<ul>
<li>所有节点的值都是唯一的。</li>
<li>p、q 为不同节点且均存在于给定的二叉搜索树中。</li>
</ul>
<h2>解题思路</h2>
<p>平衡二叉树的每个节点值都满足小于右子树而大于左子树。所以，一共有三种情况。</p>
<ul>
<li>如果两个 node 中都大于 root.val，则答案一定在右子树</li>
<li>如果两个 node 都小于 root.val，则答案一定在左子树</li>
<li>都不是的话，则返回 root（即两个 node 在左右，或者与 root 相等）</li>
</ul>
<h2>解法优化</h2>
<p>判断两个 node 的值与 root 的值做减法后相乘，然后用返回三元表达式结果。</p>
