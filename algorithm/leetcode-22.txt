8:T132b,<h2>题目描述</h2>
<p>给出 n 代表生成括号的对数，请你写出一个函数，使其能够生成所有可能的并且有效的括号组合。</p>
<p>例如，给出 n = 3，生成结果为：</p>
<pre><code class="hljs language-javascript">[<span class="hljs-string">"((()))"</span>, <span class="hljs-string">"(()())"</span>, <span class="hljs-string">"(())()"</span>, <span class="hljs-string">"()(())"</span>, <span class="hljs-string">"()()()"</span>];
</code></pre>
<h2>解题思路</h2>
<p>用回溯法，每当遇到左括号和右括号的数量都等于 n 时，将括号组合添加到结果中。 每次递归时，处理流程如下：</p>
<ul>
<li>判断当前右括号数量为 n 时，即匹配完成，添加结果</li>
<li>当左括号还能添加(leftCount 小于 n)，添加一个左括号</li>
<li>当右括号还能添加(rightCount 小于 leftCount 小于 n)，添加一个右括号</li>
</ul>
<h2>解法优化</h2>
<p>括号总是成对出现的，即</p>
<pre><code class="hljs language-javascript">n=<span class="hljs-number">1</span> res=[<span class="hljs-string">"()"</span>]
n=<span class="hljs-number">2</span> res=[<span class="hljs-string">"(())"</span>, <span class="hljs-string">"()()"</span>]
n=<span class="hljs-number">3</span> res=[<span class="hljs-string">"()()()"</span>,<span class="hljs-string">"()(())"</span>,<span class="hljs-string">"(())()"</span>,<span class="hljs-string">"(()())"</span>,<span class="hljs-string">"((()))"</span>]
</code></pre>
<p>新增加的一个是在上一个基础上追加 1 对(),懂是懂了，感觉是动态规划。。(渣算法，还没复习。。，学完之后再回来补坑)</p>
