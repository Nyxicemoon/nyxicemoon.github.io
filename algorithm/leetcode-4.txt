8:Tec5,<h2>题目描述</h2>
<p>给定两个大小为 m 和 n 的有序数组 nums1 和 nums2。</p>
<p>请你找出这两个有序数组的中位数，并且要求算法的时间复杂度为 O(log(m + n))。</p>
<p>你可以假设 nums1 和 nums2 不会同时为空。</p>
<p>示例 1：</p>
<pre><code class="hljs language-javascript">nums1 = [<span class="hljs-number">1</span>, <span class="hljs-number">3</span>]
nums2 = [<span class="hljs-number">2</span>]

则中位数是 <span class="hljs-number">2.0</span>
</code></pre>
<p>示例 2：</p>
<pre><code class="hljs language-javascript">nums1 = [<span class="hljs-number">1</span>, <span class="hljs-number">2</span>];
nums2 = [<span class="hljs-number">3</span>, <span class="hljs-number">4</span>];

则中位数是(<span class="hljs-number">2</span> + <span class="hljs-number">3</span>) / <span class="hljs-number">2</span> = <span class="hljs-number">2.5</span>;
</code></pre>
<h2>解题思路</h2>
<p>由于是两个排好序的数组，所以将两个数组合并，再找中位数就好了。（这难度不像是苦难啊。。）</p>
<h2>解法优化</h2>
<p>排名第一的思路相同，但由于我使用了<code>shift()</code>方法，导致速度上有所下降，改进为定义两个<code>index</code>，每次向 res 数组添加时，改为<code>index</code>的增加。</p>
<h2>代码展示</h2>
<pre><code class="hljs language-javascript"><span class="hljs-comment">/**
 * <span class="hljs-doctag">@param</span> {<span class="hljs-type">number[]</span>} <span class="hljs-variable">nums1</span>
 * <span class="hljs-doctag">@param</span> {<span class="hljs-type">number[]</span>} <span class="hljs-variable">nums2</span>

<span class="hljs-keyword">var</span> findMedianSortedArrays = <span class="hljs-keyword">function</span>(<span class="hljs-params">nums1, nums2</span>) {

  avg = <span class="hljs-built_in">parseInt</span>(avg);

  <span class="hljs-keyword">let</span> res = [];
  <span class="hljs-keyword">while</span> (nums1.<span class="hljs-property">length</span> != <span class="hljs-number">0</span> &#x26;&#x26; nums2.<span class="hljs-property">length</span> != <span class="hljs-number">0</span>) {
    <span class="hljs-keyword">if</span> (nums1[<span class="hljs-number">0</span>] &#x3C; nums2[<span class="hljs-number">0</span>]) {
      res.<span class="hljs-title function_">push</span>(nums1.<span class="hljs-title function_">shift</span>());
    } <span class="hljs-keyword">else</span> <span class="hljs-keyword">if</span> (nums1[<span class="hljs-number">0</span>] > nums2[<span class="hljs-number">0</span>]) {
      res.<span class="hljs-title function_">push</span>(nums2.<span class="hljs-title function_">shift</span>());

  <span class="hljs-keyword">return</span> flg ? (res[avg] + res[avg - <span class="hljs-number">1</span>]) / <span class="hljs-number">2</span> : res[avg];
};
