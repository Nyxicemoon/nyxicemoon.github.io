8:T111d,<h2>题目描述</h2>
<p>给定一个字符串，请将字符串里的字符按照出现的频率降序排列。</p>
<p>示例 1:</p>
<pre><code>输入: "tree"

输出: "eert"

解释:
'e'出现两次，'r'和't'都只出现一次。
因此'e'必须出现在'r'和't'之前。此外，"eetr"也是一个有效的答案。
</code></pre>
<p>示例 2:</p>
<pre><code>输入: "cccaaa"

输出: "cccaaa"

解释:
'c'和'a'都出现三次。此外，"aaaccc"也是有效的答案。
注意"cacaca"是不正确的，因为相同的字母必须放在一起。
</code></pre>
<p>示例 3:</p>
<pre><code>输入: "Aabb"

输出: "bbAa"

解释:
此外，"bbaA"也是一个有效的答案，但"Aabb"是不正确的。
注意'A'和'a'被认为是两种不同的字符。
</code></pre>
<h2>解题思路</h2>
<p>将数组转化成 key 是字母，value 是次数的 map，再根据 value 排序，然后依次打印出次数个字母。</p>
<h2>解法优化</h2>
<p>将if(map[key]) map[key] = 0;优化为map[key] = map[key] || 0;后提高了 15ms 左右。</p>
<p>reduce 和 forEach 之间也有性能上的差别．</p>
<p>第一名的思路是，将 map 按照 key 转换为次数个字母的字符串数组，再按照数组的长度排序，合并数组Object.keys(map).map((key) => key.repeat(map[key])).sort((x, y) => y.length - x.length).join('')</p>
